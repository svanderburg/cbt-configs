PCem configuration
==================
Kernel configuration I used for an emulated machine in PCem with the following
hardware:

Machine
-------
* Machine: [Slot 1] Gigabyte GA-686BX
* CPU: Intel Pentium II/450
* Memory: 128 MB

Graphics
--------
* Device: Voodoo3 

Audio
-----
* Device: Sound Blaster AWE32

Mouse
-----
* Mouse: Microsoft Intellimouse (PS/2)

Network
-------
* Network: Realtek RTL8029AS

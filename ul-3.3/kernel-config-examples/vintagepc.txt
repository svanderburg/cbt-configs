Hardware
========
This is a kernel configuration for my custom built vintage PC that has the
following hardware:

Motherboard:   ASUS P2B-F (Intel 440BX chipset, USB support)
CPU:           Intel Pentium III 450
RAM:           192 MiB
Floppy drives: A GoTek floppy emulator (primary) and a 1.44 MB 3.5" floppy drive
               (secondary)
DVD ROM:       LG DVD-ROM + CD-RW
Hard drive:    CF2IDE device with 32 MiB memory card
Keyboard:      PS/2
Mouse:         PS/2
Graphics card: Diamond Viper V770 (NVIDIA RIVA TNT2 chipset)
Sound card:    Sound Blaster Audigy 2
Network card:  RealTek 8193
Joypad:        Microsoft Sidewinder
